Mauth is a Two-Factor Authentication app with support for TOTP and HOTP and compatibility with Google Authenticator. It comes with a rich UI while also providing many necessary features.

The app offers the following features:
- Compatibility with Google Authenticator
- Security
- Multiple methods for adding accounts
- Support for various authentication algorithms
- Account organization and management
- Data export and import

Mauth is currently in a beta phase and accepts all user feedback regarding how the app looks and works.
